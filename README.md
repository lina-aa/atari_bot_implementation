## Atari KingKong bot implementation for AI in game development university course

The objective of this project is to create a bot for Atari game KingKong. 

Bot versions in this repo:
1. v1 - basic bot version; learned only to jump over bombs
2. v2 - bot with a early custom wrapper that gave additional reward for getting higher
3. v3 - bot with upgraded wrapper

Height RAM adress discovered throught experiments. It was identified as '0x21 (33)'.
